# 👤 Guía para Usuarios Finales

**Para personas SIN conocimientos técnicos que quieren usar la aplicación.**

---

## 📥 Paso 1: Descargar

Descarga la versión de Windows, Android o Linux desde el enlace de distribución. El archivo final será un `.exe` que podrás abrir con doble clic.

---

## 💾 Paso 2: Instalar

Si te entregan un instalador, ejecútalo y sigue el asistente. Si te entregan una carpeta portable, entra dentro y abre `Plant-AId.exe`.

La aplicación se conecta al servidor de la universidad, así que solo necesita que la URL de la API esté configurada correctamente. {indicar aquí la URL final}


---

## 🚀 Paso 3: Primer Uso

### Iniciar la aplicación
- **Windows**: Doble clic en el icono del escritorio
- **Mac**: Abre `Aplicaciones` → `PlantAid`

En Windows no hace falta abrir Python ni una terminal. La aplicación se abre como cualquier programa instalado.

### Primera pantalla

Si no tienes una cuenta todavía, regístrate y se te dará el rol de Usuario. Si quieres otro rol contacta con el desarrollador.

Una vez tengas tu cuenta, selecciona tu rol e inicia sesión con tus credenciales.

Si te da error de conexión asegúrate de que la dirección a la que te estás conectando es la correcta en **Configuración de conexión**

---

## 🎯 Funciones Principales para Usuarios

### 1. Subir Imágenes
- Botón "Subir Foto"
- Selecciona una imagen de una hoja
- Elige la etiqueta de tu imagen si la conoces
- ¡Listo! Se añade a la base de datos
- Posibilidad de que un modelo prediga la etiqueta de tu imagen

### 2. Ver Experimentos
- Botón "Experimentos"
- Aquí se muestran los modelos ya entrenados y sus caracterísitcas
- Puedes ver resultados de cada modelo y compararlos
- Puedes crear un nuevo experimento y solicitar su entrenamiento

### 3. Subir imágenes masivamente
- Botón "Subida masiva"
- Sube un conjunto de imágenes
- Funcionalidad delicada. Se requiere rol de Usuario+

---

## ⚠️ Requisitos para Windows

- Permitir que Windows ejecute la aplicación la primera vez si el antivirus o SmartScreen muestran un aviso.
- Si la app no conecta, revisa en "Configuración de conexión" que la URL de la API sea la correcta.


## 🎯 Funciones Principales para Etiquetadores

### 1. 🏷️ Etiquetar o validar Imágenes
- Botón "Etiquetar/Validar"
- Filtra qué imágenes quieres comprobar
- Selecciona una imagen de la lista
- Selecciona la etiqueta
- Haz clic en "Etiquetar"
- Valida si estás seguro/a de que la etiqueta es correcta

### 2. 🏷️ Editar clases
- Botón "Editar clases"
- Navega a través de las clases existentes
- Modifica la clasificación de la enfermedad o su nombre científico
- Guarda los cambios
- Añade clases si es necesario
- Proceder con cuidado y después de consultar
